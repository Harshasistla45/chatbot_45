# Groq Chatbot

A simple AI chatbot built with Python and the Groq API.

## Features
- Fast AI responses
- Terminal-based chatbot
- Easy to extend into a web app

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Add your Groq API key in the code.

3. Run the app:
   ```bash
   python chatbot.py
   ```

## Files
- `chatbot.py`
- `requirements.txt`

## Notes
This project uses the Groq API for fast LLM responses.
